💡 任务发布小贴士

  • 明确具体：任务描述要具体明确，避免模糊不清
  • 合理积分：根据任务难度和时间成本设置积分
  • 适当期限：给孩子足够但不过长的完成时间
  • 及时确认：孩子提交后要及时确认任务完成情况
  • 积极鼓励：多给予正面反馈和鼓励

⭐ 积分设置建议

5-10 积分
简单任务
如整理书桌、洗碗等
10-20 积分
中等任务
如完成作业、读书等
20-30 积分
困难任务
如深度学习、复杂项目等
